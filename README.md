# Lirionex — Global Financial Platform Demo

> Protótipo interativo da plataforma financeira Lirionex: Gateway de Pagamentos, IB Network, Trading MT5, Blockchain Ledger e KYC/Compliance.

## 🚀 Demo ao vivo

👉 **[lirionex-demo.vercel.app](https://lirionex-demo.vercel.app)**

## 🔐 Acessos de demonstração

| Perfil | Login |
|---|---|
| Master IB | master@lirionex.com |
| IB Nível 1 | ib@lirionex.com |
| Trader | trader@lirionex.com |

> Qualquer senha funciona no ambiente demo.

## 📋 Funcionalidades do protótipo

- **Dashboard** — Saldo, volume, comissões, gráfico mensal
- **Gateway Pix** — Depósito com câmbio BRL→USD + QR Code mockado
- **Trading MT5** — Posições abertas, P&L em tempo real
- **IB Network** — Árvore da rede multinível (Master IB → IB → Traders)
- **Comissões** — Extrato, metas, smart contract automático
- **Blockchain Ledger** — Feed de transações on-chain (Polygon)
- **KYC / Compliance** — Etapas de verificação e nível de acesso

## 🛠 Stack

- HTML5 + CSS3 + JavaScript (vanilla, sem dependências)
- Deploy: Vercel (static)
- CI/CD: GitHub → Vercel (auto-deploy em cada push)
